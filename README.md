## Install

- Copy the `zzz-dictquicklookupremake.koplugin` folder to your KOReader `plugins` directory on the device.
- Copy the icons into the equivalent folder in KOReader. E.g. `/koreader/resources/icons/mdlight`
- Restart KOReader.
